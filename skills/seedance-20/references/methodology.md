# Anban Seedance Methodology

Use 素材角色分配 before prompt writing. Assign 主体身份, 产品外观, 场景背景, 首帧, 尾帧, 运镜, 节奏, 音色, BGM, and 字体/文字风格 to either user media, generated visual anchors, project profile, or prompt-only guidance. Common business modes include 个人 IP, 高效段子, 创作定位, 参考视频复刻, and 视觉锚定图.

Visual anchor count is explicit: 0 张 when references already cover identity/product/first frame, 1 张 for one stable subject, 2 张 for person plus product, 3 张 only for multi-segment or first/tail frame control.

Director Formula: subject + scene + action + one visible beat + one primary camera move + timing + audio + style. Use anti-slop and a fidelity budget: reduce prompt density when identity/product fidelity matters.
